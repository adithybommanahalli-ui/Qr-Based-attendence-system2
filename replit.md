# Replit App

## Overview
A simple Node.js + Express web application.

## Tech Stack
- **Runtime**: Node.js 20
- **Framework**: Express
- **Port**: 5000

## Project Structure
- `server.js` — Express server entry point
- `public/index.html` — Static frontend
- `package.json` — Node.js dependencies

## Running the App
The app runs automatically via the "Start application" workflow.
It listens on `0.0.0.0:5000`.

## Development
To make changes, edit `public/index.html` for the frontend or `server.js` for server logic, then restart the workflow.
